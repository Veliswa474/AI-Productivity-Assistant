# ProductivityAI – AI Workplace Productivity Assistant

## 📌 Project Overview

ProductivityAI is a modern AI-powered workplace productivity assistant designed to help professionals automate repetitive daily tasks using Artificial Intelligence. The application improves workplace efficiency by generating professional emails, summarizing meeting notes, planning tasks, conducting research, and providing an intelligent AI chatbot assistant.

This project was developed as part of the **CAPACITI AI Skills Accelerator Programme** and demonstrates practical AI implementation, prompt engineering, responsible AI practices, and modern web application development.

---

# 🎯 Problem Statement

Professionals spend a significant amount of time completing repetitive administrative tasks such as writing emails, organizing schedules, summarizing meetings, and researching information.

ProductivityAI solves this problem by providing AI-powered tools that automate these tasks, allowing users to work more efficiently while improving productivity.

The application aims to:

# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/productivity-ai.git
```

Navigate into the project

```bash
cd productivity-ai
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```
